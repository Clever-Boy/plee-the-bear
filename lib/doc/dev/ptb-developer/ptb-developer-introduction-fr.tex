\section{Introduction}
L'objectif de ce document est de présenter l'ensemble des classes et
méthodes utilisées pour le jeu \emph{Plee the Bear} (PTB). Le scénario
est sûrement décrit dans un document joint.

Toutes ces classes utilisent intensivement la classe \verb|claw::logger|, qui
devra donc être initialisée au début du programme. On implémentera le programme
principal à l'aide de la classe \verb|claw::application|, ce qui nous
simplifiera la tâche.
